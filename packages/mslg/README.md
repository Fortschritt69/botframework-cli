# Usage
<!-- usage -->
```sh-session
$ npm install -g mslg
$ mslg COMMAND
running command...
$ mslg (-v|--version|version)
mslg/1.0.0 win32-x64 node-v10.16.0
$ mslg --help [COMMAND]
USAGE
  $ mslg COMMAND
...
```
